## Local Setup:

Medusa Backend (by default) runs at localhost:9000

`cd` into `medusa-store-server` and run `../../node_modules/.bin/medusa develop`.

Medusa frontend

`cd` into `medusa-store-frontend`, install deps and run `npm run dev`
